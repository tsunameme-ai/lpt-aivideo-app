## Get API Keys 
### ModelsLab
1. Signup [modellabs]('https://modelslab.com/')
2. Create key [here]('https://modelslab.com/dashboard/apikeys')

## Run the app
1. `npm install`
2. Create `.env` file with contents from `env-example` file.
